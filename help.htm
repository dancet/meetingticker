Help

If you're having problems with Meeting Ticker, best to reload the page, start again and make sure you're using the following format:

<time in mins><space><agenda item text>

The format is very important and you'll have all sorts of problems if you aren't following the guide above. This is a sample agenda:

5 Welcome and introduction of attendees
10 Discussion of latest report and feedback
3 Update from Barry on the office move next week

The input above will create three agenda items with durations of 5, 10 and 3 minutes respectively. The important parts are the spaces between the digits and the agenda item and the new line after the item.

The save button sends your agenda to bit.ly which returns a shortened URL. This is a neat way to save the agenda without having to save stuff to a database. If you've got concerns about the privacy of your data, the only thing that is saved is your agenda text. No IP addresses or any of that stuff is saved anywhere. The site is built entirely with HTML, JavaScript and CSS so check the source if you're super paranoid.
